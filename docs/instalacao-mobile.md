# Instalacao no celular

Link publico atual:

`https://lhgcampos.github.io/bolao2026/`

## iPhone

1. Abra o link no Safari.
2. Toque em `Compartilhar`.
3. Toque em `Adicionar a Tela de Inicio`.

## Android

1. Abra o link no Chrome.
2. Toque em `Instalar app`.
3. Se nao aparecer, abra o menu e use `Adicionar a tela inicial`.

## Observacoes

- O app funciona direto no navegador, mesmo sem instalar.
- A instalacao nao passa por App Store nem Google Play.
- Depois de instalar, o Bolao 2026 abre pela tela inicial como app.
